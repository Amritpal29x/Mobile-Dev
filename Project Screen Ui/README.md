Project Overview

This project is a responsive Notes Application built using React Native and Expo.
The app allows users to:

Create notes
Edit notes
Delete notes
Search notes
Toggle Dark Mode / Light Mode
Experience responsive layouts on phones and tablets

The UI is designed with clean spacing, responsive layouts, and modern styling.

Features
Notes Management
Create new notes
Edit existing notes
Delete notes
Search/filter notes instantly
Theme Support
Dark Mode
Light Mode
Dynamic UI color changes using React state
Responsive Design
Supports:
Mobile phones
Tablets
Landscape mode
Portrait mode
Orientation Handling
Tablets automatically switch to landscape
Phones remain in portrait mode
React Native Components Used
FlatList
TextInput
Pressable
Switch

ImageBackground
ScrollView
View
Text
React Hooks Used

useMemo()
useWindowDimensions()
Styling Techniques Used
StyleSheet.create()
Responsive dynamic styling
Flexbox layouts
Elevation and rounded cards
Conditional theme styling
Additional Improvements
Floating Action Button
Responsive tablet layout
Search functionality
Context delete menu
Dark mode UI optimization
Landscape support for tablets
Smooth UI spacing and typography
